What is ThumbFish

 

ThumbFish is a simple utility for chemists and people working with chemical files. It integrates with Windows and provides molecule Thumbnail, Preview, Search and other useful features. It is a set of Windows Shell Extension handlers developed to quickly identify and analyze structure files. ThumbFish urrently supports the following chemical structure files:

  • MOL (MDL Molfile Format - V2000 and V3000)
  • RXN (Reaction file - V2000 and V3000)
  • SMILES (Simplified Molecular Input Line Entry Specification)
  • SMARTS (SMiles ARbitrary Target Specification)
  • SDF (Structure Data File - V2000 and V3000)
  • RDF (Reaction Data File - V2000 and V3000)
  • CML (Chemical Markup Language)

The handlers makes it easy to identify and search these chemical files without opening the associated application. These handlers are NOT dependent on any specific commercially available Chemistry application. It uses the open source organic chemistry toolkit named Indigo by GGA Software. The other open source component used in ThumbFish is Pantheios logging library.

ThumbFish contains the following Shell Extension Handlers for Windows OS:

  • Thumbnail Handler: Displays thumbnails for supported formats to easily identify structure files
  • Preview Handler: Displays preview and properties for selected file
  • Search Handler: Searches for a structure based on its properties
  • Context-Menu Handler: Provides additional useful options on structure files

Using ThumbFish

Thumbnail Handler

Thumbnail handler displays thumbnail images in Windows Explorer for supported chemical formats. In case of SDF files, the thumbnail shows the first four structures to help users identify a particular SDF without opening the file in the associated program.




 

Preview Handler

Preview handler can be used to view common properties associated with a structure file. Preview shows properties such as Name, Formula, SMILES, InChI and many more. These properties can be copied to the clipboard using the "Copy Properties" option in the context-menu on the preview dialog. It copies all available properties to the clipboard in a Key:Value format.
Note that no properties are displayed for multi-molecule files such as SDF.

If you do not see previews in Windows Explorer then try the following:
  1. Enable Preview Handlers by checking the following option: [Windows Explorer Tools >> Folder Options >> View >>Show preview handlers in preview pane]
  2. To Show/Hide preview pane: [Windows Explorer >> Organize (Extreme Left on toolbar) >> Layout >>Preview Pane]

Preview window provides a context-menu that has different useful options to Save and Copy structure and properties.
  • Save Structure: Saves a structure to disk in different formats: PNG, PDF, SVG, EMF and CDXML.
  • Copy Properties: Copies the currently displayed structure properties to clipboard in Key:Value format
  • Copy Structure As: Copies the current structure to clipboard in different useful formats including MDL MOL and ChemDraw's CDXML


Search Handler

IFilter search handlers are used by Windows Search Engine to index information inside proprietry files. This IFilter search handler indexes properties such as formula, atom count, mass etc inside supported files and returns them to the Windows Search Engine so that the user can search for structures with given properties. Normally, the search engine only searches for file names and not inside a file's contents but the search handler makes it possible to search by a molecule properties. For example if there are 100 structure files inside a folder then typing benzene will search for file names containing the word 'benzene' and matching files will be returned. With a Search handler such as this one, a user can search for structures having atom count as 12 by typing na_12.

Setting Up Search

  • Make sure Windows Search is enabled in Control Panel [Control Panel >> Programs and Features >> Turn Windows Features on or off >> Windows Search]
  • Make sure that the folder in which you want to search is included for indexing [Control Panel >> Indexing Options] (your folder should be listed here)

Troubleshooting Search

You can try the following steps if search is not working as expected.
  • Rebuild the search index in case it is corrupt or not built because of user activity: [Control Panel >> Indexing Options >> Advanced >> Rebuild] Note that this can take some time besed on the number of files to index
  • Enable to always search a file's content [Explorer >> Folder Options >> Search >> What to Search >> Always search file name and contents]
  • Finally to always reindex and search: [Explorer >> Tools >> Options >> Search >> Don't use the index when searching in file folders...]
  • Still not working, check the ThumbFish Log for any error messages.

Search Examples

Note that a underscore '_' is used instead of an equals to '=' sign because an equals to sign is a word breaker. Properties are supplied to the search engine as a long string containing semicolons to separate different properties ('formula_c6h6;na_12;name_benzene;'). These properties are further split by the search engine to extract words that can be indexed so passing a 'na=12' will store 'na' and '12' because '=' is a word breaker. In order to make search such as 'na_12' possible it should not contain any word breaker.The following property searches are possible:
  • Name: name_{x} [e.g. n_benzene] Note that name is the structure name stored in the file
  • Formula: formula_{x} [e.g. formula_c6h6]
  • NumAtoms: na_{x} [e.g. na_12]
  • NumBonds: nb_{x} [e.g. nb_10]
  • Molecular Weight: mweight_{x} [e.g. mweight_207]
  • Implicit Hydrogens: impl_{x} [e.g. impl_4]
  • Heavy Atoms: heavya_{x} [e.g. heavya_2]
  • Most Abundant Mass: mamass_{x} [e.g. mamass_106]
  • Mono Isotopic Mass: mimass_{x} [e.g. mimass_210]
  • Chirality: chiral_{x} [e.g. chiral_yes]
  • Has Coordinates: hascoord_{x} [e.g. hascoord_yes]
  • Has Z Coordinates: haszcoord_{x} [e.g. haszcoord_no]

Context-Menu handler

ThumbFish context-menu is integrated into the Window Explorer context-menu and provides commands that work on single and multi molecule files. As of now the context-menu does not have any feature commands. It is there to let users know that the future versions of this product will add commands to it.

Known Limitations

  1. Copying structure as Enhanced Metafile (EMF) Picture does not display atom labels.
  2. When a structure is saved using the "Save Structure" command in Preview Handler then it can only be saved in user's AppData\LocalLow directory. This is because the Preview Handler runs in a low integrity mode under Windows and does not have access to other locations.
  3. The Copy Structure As >> InChi Key command is NOT operational. The command has been intentionally disabled because of some problems associated with it.
  4. ThumbFish handles do not currently support Windows 8. Some of the APIs used in the handlers fail on Windows 8 which is being investigated.
